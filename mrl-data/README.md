# mrl-data — 農藥殘留容許量(MRL)資料管線

**此資料夾目前完全不影響 App。** 尚未接入 `index.html`,不在 Service Worker 快取,
不會被使用者下載。屬 MRL 對照功能的**階段 0**(見 `docs/MRL對照實施手冊.md`)。

## 檔案

| 檔案 | 用途 |
|---|---|
| `fetch-mrl.mjs` | 下載腳本 |
| `mrl-<日期>.json` | 帶日期的資料快照(保留,可比對版本差異) |
| `latest.json` | 最新快照(其他腳本讀這個) |
| `build-review-list.mjs` | 產出需人工確認清單 |
| `待人工確認.md` | **人工校對用**,實作前必須完成 |

## 使用

```text
node mrl-data/fetch-mrl.mjs          # 下載最新 MRL 資料
node mrl-data/build-review-list.mjs  # 重新產出待確認清單
```

`fetch-mrl.mjs` 會驗證回傳為非空陣列且欄位符合預期,不符即中止,
避免把壞資料覆蓋上去。

## 資料來源

- 政府資料開放平台 dataset **8944**「農藥殘留容許量標準」(衛福部食藥署)
- `https://data.fda.gov.tw/data/opendata/export/13/json`
- 欄位:`國際普通名稱` / `普通名稱` / `作物類別` / `容許量ppm` / `備註`

法源:《農藥殘留容許量標準》「**表中未列者均不得檢出**」。

## 重要原則

1. 與農藥登記資料**分開維護、分開標版本**,不併入 `index.html`。
2. 官方資料有約 9.3% 缺字(罕用字變 `?`),**萬用字元比對只可作候選提示**;
   一對多歧義者(如 `亞??` → 亞醌蟎/亞滅寧/亞培松)未經人工確認**不得採用**。
3. 生物製劑類查無容許量,**可能本就免訂**,不可逕標為「未訂容許量」。
4. 帶星號的容許量(如 `0.01*`)保留原字串,不做數值運算。
5. 未確認、對照失敗者一律「⚪ 無法對照」,**不得**顯示綠色標示。
