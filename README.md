# 噴前查

台灣農友的農藥登記查詢、配藥計算與施藥紀錄工具。

正式網站：https://xiao68738-hash.github.io/searchbefore/

## 新手版工作方式

1. `main` 是正式網站，不直接修改。
2. 每次工作從 `main` 建立一個分支，例如 `fix/safety-update`。
3. 在分支修改並執行測試。
4. 建立 Pull Request，確認後才合併回 `main`。
5. 合併完成並確認網站正常後，刪除已完成的分支。

## 執行安全測試

不需要安裝額外套件，只要電腦有 Node.js：

```text
node tests/run-all.js
```

## 發布前檢查

- 粉劑只顯示公克，不得換算瓶蓋或湯匙。
- 採收備註中的設施栽培天數不得被忽略。
- 同一田區或種植批次的所有施藥紀錄都要納入，取最晚可採日。
- 任一紀錄缺少採收期時，不得顯示「可採收」。
- 更新 `index.html` 後同步更新 `sw.js` 的 `CACHE_VERSION`。
- `private_key.pem`、其他 `.pem` 私鑰檔不得上傳。

## 重要安全聲明

本工具僅供查詢與自主紀錄。實際農藥使用、稀釋倍數與安全採收期，仍以產品標示及主管機關最新公告為準。
