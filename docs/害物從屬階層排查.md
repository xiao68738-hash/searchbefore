# 害物從屬階層排查清單

更新日期:2026-07-19
資料版本:2026-07(農業部農藥登記開放資料)
狀態:**排查報告,尚未實作**
起因:農友回饋夜蛾歸類問題;經全面排查發現不只夜蛾,是**全資料庫的結構性問題**。

## 一、問題本質

資料庫的病蟲害名稱同時存在**群組**(如「夜蛾類」)與**具體物種**(如「斜紋夜蛾」),兩者在同一作物下是**各自獨立的條目**,藥劑清單也各自獨立。

農友田裡看到斜紋夜蛾 → 點「斜紋夜蛾」→ 只看到 1 種藥;
但同一作物的「夜蛾類」底下另有 33 種登記藥劑,他**完全看不到**。反之亦然。

這不是資料錯誤(登記本來就分開登),而是**查詢動線的可發現性問題**。

## 二、排查結果總覽

- 害物名稱總數:**286**,其中「⋯類」群組:**43**
- 同一作物內並存的從屬配對:**42 組**(strong,判定明確)
- 其中**藥劑完全不重疊者:18 組** ← 農友最容易漏看的情形
- 另發現**群組之間也有階層**(如 介殼蟲類 ⊃ 粉介殼蟲類/盾介殼蟲類/軟介殼蟲類)

## 三、判定方法與可信度

**採用(strong)**:子項名稱**完整包含**群組字根(「斜紋夜蛾」⊃「夜蛾」)。42 組經逐一人工複核,無誤判。

**排除(weak)**:僅**末字相同**者(120 組)全數不採用,因人工判讀發現絕大多數是誤判:

| 誤判例 | 為何是錯的 |
|---|---|
| 毒蛾類 ⊃ 斜紋夜蛾 | 斜紋夜蛾屬夜蛾科,非毒蛾;只因都以「蛾」結尾 |
| 螟蛾類 ⊃ 小菜蛾 | 小菜蛾屬菜蛾科,非螟蛾 |
| 根蟎類 ⊃ 二點葉蟎 | 二點葉蟎是葉蟎,非根蟎 |
| 蚜蟲類 ⊃ 鱗翅目害蟲 / 根瘤線蟲 | 只因都以「蟲」結尾,完全無關 |

**結論**:純字串規則不足以判定生物分類從屬,必須人工複核後才可採用。

## 五、已知漏網(命名不含字根,字串規則抓不到)

以下屬**真實從屬但名稱無共同字根**,需昆蟲分類專業確認後才可納入,目前**不主張自行加入**:

| 疑似從屬 | 說明 |
|---|---|
| 夜蛾類 ⊃ **秋行軍蟲** | 秋行軍蟲(*Spodoptera frugiperda*)與斜紋夜蛾同屬 *Spodoptera*,屬夜蛾科 |
| 夜蛾類 ⊃ **切根蟲** | 切根蟲(地老虎)為夜蛾科幼蟲的通稱 |
| 介殼蟲類 ⊃ **膠蟲** | 膠蟲屬介殼蟲總科 |

**處理原則**:這類需要專業判斷者,在取得權威依據(如植物保護手冊、防檢署分類)前不納入對照表。寧可少對照,不可錯對照。

## 六、建議做法(尚未實作)

比照現有作物群組的處理方式(藥劑卡片已有 `登記於:上層群組` 徽章),在害物層做**可發現性補強,但不自動合併**:

1. 選擇具體害物(如斜紋夜蛾)時,另開一個**明確標示的區塊**:「同作物『夜蛾類』另有 N 種登記藥劑」,可展開查看。
2. 反向亦同:選群組時提示底下具體物種另有登記。
3. **不靜默合併**兩者清單——理由同交接文件安全規則 7(分類包含關係未經官方逐項校驗前不得自動合併上層登記)。
4. 對照表比照 `crop-forms.js` 作法:規則寫成獨立模組,並以測試驗證每組從屬的字根關係實際成立。

## 七、已知限制
- 本排查僅依**名稱字串**,不含生物分類資料庫;第五節的漏網即為此限制的直接後果。
- 群組登記是否法律上涵蓋所有子物種,屬主管機關解釋範圍,本工具不做認定。
- 排查基於 2026-07 版資料;資料更新後需重跑。
