# 作物收穫型態消歧 — 全作物排查與變更說明

更新日期:2026-07-19
資料版本:2026-07(農業部農藥登記開放資料)
起因:農友回饋「青蒜跟蒜頭分不出來(小葉菜類 vs 根莖菜類)」「乾豆跟豆菜類要確認差異」。

## 排查方法

以程式全面掃描資料庫(284 個作物、16,342 筆藥劑)的備註欄,比對「適用於X」「限採收X者使用」「禁止/不得使用於X」「X採收前N天」等限制語彙,再**逐一人工判讀**每個命中作物,分成三類處理。掃描與比對腳本、每條規則的原文出處均可重跑驗證(tests/crop-forms.test.js 會驗證每條規則文字實際存在於該作物備註)。

排查結果:**69 個作物備註含限制語彙**,其中:

- **18 個作物確認有「收穫型態歧義」→ 新增型態選擇**
- **13 個作物為單側標註或類別層備註 → 不變更**(備註原文已顯示於卡片,理由見下)
- **38 個作物僅為施用時機限制(如限開花前、僅限苗期)→ 不屬型態問題,不變更**

### 分組規則(保守原則)
1. **只依備註明確文字**分類或排除;每條規則文字均經測試驗證實際存在於該作物備註。
2. **未註明型態的藥劑不會自動歸類**:
   - 一般型態(如蒜頭、鮮豆莢):未註明者留在主清單,標「未註明型態」。
   - **strict 型態**(青蒜/蒜葉、蕗蕎葉、珠蔥葉、毛豆、葉用豌豆、碧玉筍、枸杞葉、葉用甘藷、葉用甜菜——這些屬另有獨立商品類別的次要部位):母作物未註明的登記**不可推定適用**,移入「待確認」摺疊區並附理由。
3. **任何藥劑都不會被移除**:不符/待確認者收在可展開的摺疊區,附排除理由,仍可完整查看與操作。測試強制驗證「符合+未註明+排除 = 全部」且索引不重排(帶入計算/紀錄一定對到同一筆)。
4. **不選型態=行為與改版前完全相同**(全部顯示),測試把關。

### 新增搜尋別名(23 個,只收對應無疑義者)
青蒜/蒜苗→蒜(蒜葉)、蒜頭→蒜(蒜頭)、毛豆→大豆(毛豆)、黃豆→大豆(乾大豆)、黑豆→大豆(不預選)、豌豆苗/豌豆嬰→豌豆(葉用)、碧玉筍/金針花→金針、紅蔥頭→珠蔥、蕎頭→蕗蕎、地瓜葉/甘藷葉/葉用甘藷→甘藷(葉用)、枸杞葉→枸杞、韭菜/韭黃/韭菜花→韭、蓮子/蓮藕→蓮、敏豆/四季豆→菜豆(不預選)。

**刻意不收**:「豆苗」(可能指豌豆苗、綠豆芽等多種,無法無疑義對應)。

## 二、評估後「不變更」的作物與理由

| 作物 | 命中語彙 | 不變更理由 |
|---|---|---|
| 刀豆/扁豆/翼豆/鷹嘴豆 | 適用於鮮豆(47/47/47/8) | 單側標註(僅鮮豆),乾豆側無明確標註藥劑,設型態選擇會產生空集合誤導;備註原文已顯示 |
| 紅豆/綠豆/落花生 | 適用於乾豆(11/27/13) | 同上(僅乾豆側);且該作物市場慣行即為乾豆 |
| 茶 | 適用於茶葉(25) | 茶僅採收葉部,無歧義 |
| 水稻 | 適用於水稻育苗(1) | 單筆、非收穫部位歧義 |
| 婆羅門參 | 適用於婆羅門參葉(1) | 單筆,葉用極少見,備註已顯示 |
| 甘藍 | 「適用於包葉」 | 逐字核對原文為「僅適用於**包葉菜類**」,是類別說明非部位——初掃誤判,已排除 |
| 蔥科小葉菜類/蔥科根菜類 | 不得使用於韭黃、韭菜花及蒜苔(各4) | 類別型作物,型態空間橫跨多種作物無法定義;備註原文已顯示 |
| 豆科小葉菜類/豆科豆菜類/豆科根菜類/豆科乾豆類 | 禁止使用於豆苗(與乾豆類) | 同上 |
| 瓜果類 | 禁止使用於河床地西瓜(9) | 屬地點限制,非收穫型態 |

## 三、僅施用時機限制(不屬本題,無變更)

「僅限苗期使用」「限開花前/種植前/播種前/萌芽前/結莢前/中果期前/花苞形成前/第一次開花前使用」「限水田使用」「禁止使用於設施栽培」等出現於約 38 個作物(草莓、番茄、芹菜、胡蘿蔔、荔枝、菊等)——這些是**施藥時機**限制,收穫物沒有歧義,備註原文照常顯示於藥劑卡片,不做型態處理。(設施/露天採收期差異已由既有的備註採收期解析規則「採較長天數」處理。)

## 五、已知限制
- 型態判讀完全依賴備註文字;登記資料若未寫明,本工具不推測。
- 「待確認」不代表不可用,代表登記資料未寫明——請以產品標示與官方公告為準。
- 型態選擇目前僅影響查詢顯示,尚未寫入施藥紀錄(未來若寫入需連動田區資料結構,另案評估)。
- 殘留容許量(MRL)對照為下一階段,與本次無關。
